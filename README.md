# Hay Cartitas

Hay Cartitas es una aplicación web comunitaria para reportar en qué tiendas hay cartitas del Mundial.

## Funciones actuales

- Mapa real con marcadores.
- Lista de tiendas.
- Filtros por disponibilidad.
- Agregar tiendas nuevas.
- Elegir ubicación en el mapa.
- Reportar disponibilidad.
- Historial de reportes.
- Estampas de tiempo relativas.
- Sistema de confirmación comunitaria.
- Filtro de reportes confiables.

## Estado del proyecto

Esta es una primera versión local hecha con:

- HTML
- CSS
- JavaScript
- Leaflet
- OpenStreetMap
